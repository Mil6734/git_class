Программа для создания случайного пароля

Эта программа позволяет пользователю создать случайный пароль с заданными параметрами, такими как количество символов, использование букв верхнего регистра, цифр и специальных символов.

Выбор параметров пароля:

  • Количество символов: Пользователь выбирает желаемое количество символов для пароля.
  • Включить буквы верхнего регистра: Пользователь выбирает, хочет ли он включить в пароль буквы верхнего регистра.
  • Включить цифры: Пользователь выбирает, хочет ли он включить в пароль цифры.
  • Включить специальные символы: Пользователь выбирает, хочет ли он включить в пароль специальные символы.

Генерация случайного пароля:

Программа использует указанные параметры для генерации случайного пароля.

Вывод пароля:

Сгенерированный пароль отображается пользователю.
